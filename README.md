# 🎓 Student Record Management System (Using Arrays in Java)

## 📌 Overview

The **Student Record Management System** is a console-based Java application that manages student data efficiently using the **array data structure**. This project demonstrates how arrays can be applied in real-world scenarios to store, retrieve, and manipulate structured data such as student records.

---

## 📖 Abstract

This system is designed to organize student information such as **ID, name, and marks** using arrays. Educational institutions deal with large volumes of student data, and managing it manually can lead to inefficiency and errors.

Arrays provide a simple and effective way to store multiple records sequentially and access them using index-based operations. The system allows users to perform operations like **adding, deleting, updating, and searching student records** through a menu-driven interface.

## ❗ Problem Statement

Managing student records manually or without proper structure leads to:

* Difficulty in handling large data
* Time-consuming searches
* Data duplication and errors
* Inefficient updates and deletions
* Poor data organization

This project solves these problems using arrays, ensuring **efficient storage, quick access, and organized data management**.

## ▶️ How to Run the Project

1. Install Java (JDK 8 or above)
2. Clone the repository:

   ```bash
   git clone https://github.com/your-username/StudentRecordManagementSystem
   ```
3. Navigate to the project folder
4. Compile the program:

   ```bash
   javac StudentManagementSystem.java
   ```
5. Run the program:

   ```bash
   java StudentManagementSystem
   ```
